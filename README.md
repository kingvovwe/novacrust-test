# Crypto Checkout Page Test For NovaCrust

I Vovwe, made a responsive, crypto checkout frontend built for the Novacrust assessment, based on the figma design provided below:

https://www.figma.com/design/FRfbMHys4JINX4V9qBxgbf/Frontend-Assessment?node-id=0-1

## Tech Stack
- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **State Management:** React Hooks

## Live Link
https://vovwe-novacrust-test.vercel.app/

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/kingvovwe/novacrust-test.git

2. Install dependencies:
   ```bash
   npm install

3. Run the development server:
   ```bash
   npm run dev

4. View the Results
